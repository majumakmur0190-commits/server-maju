Import Excel → tabel barang (Generate SQL / Upload)

Mapping kolom: pilih kolom Excel yang sesuai untuk setiap field tabel barang.

Catatan penting:
  • Fungsi generate & upload mengasumsikan setiap baris Excel sudah berisi field: barang_id, nama_barang, kategori_id, harga_hna, diskon_beli, diskon_jual, stok, satuan, aktif, created_at — jika tidak, gunakan mapping di atas.
  • Upload ke server mengirim data sebagai JSON ke import.php. Pastikan server dan import.php aman dan tidak mengeksekusi SQL mentah dari client tanpa validasi.
  • Jika ingin langsung menjalankan .sql di database, impor file .sql melalui tool database Anda (phpMyAdmin, MySQL CLI, dsb.).
